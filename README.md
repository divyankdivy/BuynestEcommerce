# Buynest

Buynest is a Spring Boot e-commerce application deployed on AWS. It allows users to browse products, manage carts, place orders, and handle authentication. Admins can manage products, categories, and view orders.  

## Features

- User registration, login, and profile management
- Product catalog with categories
- Cart and checkout system
- Payment integration
- Admin panel to manage products, categories, and orders
- Password reset with OTP
- Responsive front-end using Thymeleaf and CSS

## Tech Stack

- Java 17
- Spring Boot
- Spring Security
- PostgreSQL
- Thymeleaf (HTML templates)
- CSS
- AWS (for deployment)
